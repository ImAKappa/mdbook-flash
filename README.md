# mdbook-flash

An mdbook preprocessor for incorporating flashcards and fill-in-the-blanks into your documents

## Usage

Fill-in-the-blanks

```md

# My Fill Note

?`{Mitochondria} are the powerhouse of the cell!`

```

Flashcard

```md

# My Flash Note

??`{-emia} means presence in blood`

```

## Installation

```bash
mdbook-flash install
```
